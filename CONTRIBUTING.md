# Contributing



Thank you for considering a contribution to Amulet Utility Plugins.



Contributions may include code, bug reports, feature requests, testing notes, documentation improvements, screenshots, examples, or compatibility information.



## Before Contributing



Before opening an issue or pull request, please:



1. Search existing issues to check whether the same topic was already reported.
2. Use the correct issue template when possible.
3. Include relevant version information when reporting bugs.
4. Keep reports clear, practical, and focused.



## Code Contributions



When contributing code, please try to:



1. Keep changes focused.
2. Avoid unrelated rewrites.
3. Explain what changed and why.
4. Test the plugin when possible before submitting.
5. Mention the Amulet version and Minecraft version used for testing.
6. Avoid adding unnecessary dependencies.



## Project Direction



The project maintainer may reject, edit, reorganize, delay, or remove contributions when needed for compatibility, maintainability, safety, project direction, or code quality.



Not every request or contribution will be accepted.
